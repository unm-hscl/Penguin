This folder \Penguin is broken up in the following ways.

\Penguin\Matlab has all the analysis that was done through Matlab. There will be another readme within that folder
to provide further direction

The \Penguin folder contains Penguin Completion Times.xlsx . This excel spreadsheet shows the various runtimes 
for Experts 01-05 and for Novices 01-05. You will see a column for Expert01 (Interp Scaling Factor) this shows
the scaling factor for the various interpolation techniques used throughout the \Matlab analysis. 

Penguin_Trajectories shows the raw trajectories of novices vs experts.

Sooyung Saved Dataset 03132020 is the raw data set that was provided by Sooyung from Purdue University.
